## 前言
此事件发生于2019年12月3日凌晨，正主UZ使用“个人账号”YYUT搬运发布了一个极度引战的动态，其后进行的广泛的讨论

主要争议点

1.使用yyut搬运作为名字有蹭热度嫌疑，虽然正主自称为个人号(后已改名)

2.声称Overidea运营，vtb张京华存在黑料，但吃瓜众要求出示黑料，正主及其伙伴抛出了你们自己去找啊这种诱导群众当挖掘机的话语

3.口气狂妄，架子极大，把自己当成皇帝，甚至aoe大量路人群众，吃瓜群众和其他vtb粉丝
